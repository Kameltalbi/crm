## Premiers pas

### Créer un compte

1. Allez sur [crmtunisie.tn](https://crmtunisie.tn)
2. Cliquez sur "Essai gratuit"
3. Remplissez le formulaire d'inscription
4. Vérifiez votre email

### Première connexion

1. Connectez-vous avec votre email et mot de passe
2. Complétez votre profil
3. Créez votre organisation
4. Ajoutez vos premiers clients

## Gestion des clients

### Ajouter un client

1. Allez dans "Clients"
2. Cliquez sur "Nouveau client"
3. Remplissez les informations :
   - Nom de l'entreprise
   - Contact principal
   - Email
   - Téléphone
   - Adresse
4. Cliquez "Enregistrer"

### Modifier un client

1. Allez dans "Clients"
2. Cliquez sur le client
3. Cliquez sur "Modifier"
4. Mettez à jour les informations
5. Cliquez "Enregistrer"

## Gestion des opportunités

### Créer une opportunité

1. Allez dans "Opportunités"
2. Cliquez "Nouvelle opportunité"
3. Sélectionnez le client
4. Remplissez les informations :
   - Titre de l'affaire
   - Montant HT (en DT)
   - Type de prestation
   - Probabilité de réussite (0-100%)
   - Mois et année prévus
   - Statut (Prospection, Pipeline, Confirmé, Réalisé, Perdu)
5. Cliquez "Enregistrer"

### Dupliquer une opportunité

1. Allez dans "Opportunités"
2. Cliquez sur les 3 points (...) à droite de l'opportunité
3. Sélectionnez "Dupliquer"
4. Choisissez le nouveau mois et année
5. Cliquez "Dupliquer"

### Supprimer une opportunité

⚠️ **Attention** : Si le client a plusieurs opportunités, un avertissement s'affichera avant suppression.

## Lead Scoring

Le lead scoring évalue automatiquement vos opportunités de 0 à 100.

**Facteurs pris en compte :**
- Montant de l'opportunité
- Probabilité de réussite
- Statut (Pipeline = +20 points)
- Historique du client

**Interprétation :**
- 70-100 : Priorité haute
- 40-69 : Priorité moyenne
- 0-39 : Priorité basse

## Assistant IA

L'assistant IA répond à vos questions en langage naturel.

**Exemples de questions :**
- "Combien d'opportunités cette semaine ?"
- "CA réalisé cette année"
- "Actions prioritaires"
- "Breakeven"
- "Alertes opportunités à risque"

### Utiliser l'assistant IA

1. Allez dans "Assistant IA"
2. Tapez votre question
3. Cliquez sur "Envoyer"
4. L'assistant vous répond instantanément

## Templates d'emails

Créez des modèles d'emails personnalisables avec variables automatiques.

### Créer un template

1. Allez dans "Templates Emails"
2. Cliquez "Nouveau template"
3. Remplissez :
   - Nom du template
   - Sujet avec variables
   - Corps avec variables
4. Cliquez "Enregistrer"

**Variables disponibles :**
- `{client}` : Nom du client
- `{montant}` : Montant en DT
- `{date}` : Date actuelle
- `{statut}` : Statut de l'opportunité
- `{titre}` : Titre de l'affaire
- `{probabilite}` : Probabilité en %

### Utiliser un template

1. Allez dans une page détail d'opportunité
2. Cliquez "Template Email"
3. Sélectionnez un template
4. Cliquez "Aperçu" pour voir le résultat
5. Cliquez "Envoyer" pour envoyer (activité loguée)

## Facturation

### Créer un devis

1. Allez dans "Opportunités"
2. Cliquez sur les 3 points (...) de l'opportunité
3. Sélectionnez "Créer devis"
4. Le devis est généré via Softfacture

### Créer une facture

1. Allez dans "Opportunités"
2. Cliquez sur les 3 points (...) de l'opportunité
3. Sélectionnez "Créer facture"
4. La facture est générée via Softfacture

## FAQ

### Questions générales

**Q : Mes données sont-elles sécurisées ?**
R : Oui, vos données sont sauvegardées quotidiennement et hébergées en Tunisie.

**Q : Puis-je annuler mon abonnement ?**
R : Oui, vous pouvez annuler à tout moment sans engagement.

**Q : Combien d'utilisateurs puis-je avoir ?**
R : Cela dépend de votre plan : Starter (1), Pro (5), Entreprise (illimité).

### Questions techniques

**Q : Comment réinitialiser mon mot de passe ?**
R : Cliquez sur "Mot de passe oublié" sur la page de connexion.

**Q : L'application fonctionne-t-elle sur mobile ?**
R : Oui, l'application est responsive et fonctionne sur smartphone et tablette.

**Q : Puis-je importer mes données depuis Excel ?**
R : Oui, utilisez la fonction "Importer Excel" dans la page Opportunités.

### Questions sur les fonctionnalités

**Q : Qu'est-ce que le lead scoring ?**
R : C'est un système automatique qui évalue vos opportunités de 0 à 100 pour vous aider à prioriser.

**Q : Comment fonctionne l'assistant IA ?**
R : L'assistant IA analyse vos données et répond à vos questions en langage naturel.

**Q : Puis-je créer mes propres templates d'emails ?**
R : Oui, vous pouvez créer et personnaliser autant de templates que vous le souhaitez.
